O que o Hacktoberfest?

O Hacktoberfest - trazido voc pela DigitalOcean - uma comemorao do software de cdigo aberto (Open Source). Essa comemorao acontece durante o ms de Outubro inteiro. As pessoas mantenedoras so convidadas a orientar possveis pessoas colaboradoras para questes que ajudaro a levar o projeto adiante e as pessoas colaboradoras tero a oportunidade de dar retorno aos projetos de que gostam e aos que acabaram de descobrir. Nenhuma contribuio muito pequena - correes de bugs e atualizaes de documentao so formas vlidas de participao.

Desafio de 2020

A cada ano h um desafio para deixar as coisas um pouco mais "animadas". Esse ano o desafio ser: estar registrado no site oficial e enviar 4 pull requests entre o dia 1 e 31 de Outubro. Abaixo vai alguns pontos que vale a pena lembrar:

  • Todes so bem vindes! - Seja voc uma pessoa experiente na rea ou que est comeando agora voc pode participar!
  • Qualquer* contribuio para projetos no Github conta! - Contanto que o projeto seja pblico e que no seja um projeto seu.
  • "A quantidade divertida mas a qualidade a chave"! - Conseguir realizar vrias contribuies muito bom, conseguir realizar contribuies significativas melhor ainda.

Pequenas regras de participao

Para obter uma camiseta, voc deve fazer quatro pull requests (PRs) entre 1 e 31 de outubro em qualquer fuso horrio. As contribuies podem ser feitas para qualquer repositrio pblico no GitHub, no apenas para aqueles destacados. *Se uma pessoa mantenedora relatar sua contribuio como spam, isso no ser contabilizado para sua participao no Hacktoberfest. Se uma pessoa mantenedora relatar um comportamento que no esteja de acordo com o cdigo de conduta do projeto, voc no poder participar. Este ano, os primeiros 75.000 participantes podem ganhar uma camiseta. Confira todos os detalhes no site oficial.

Sobre o evento

Dessa vez o nosso encontro ser online! Iremos conversar com pessoas que so referncias quando o assunto Open Source e tambm estaremos todes ali para fazermos nossas contribuies.

No se preocupe caso voc nunca tenha feito nenhuma contribuio Open Source, esteja comeando agora na rea ou qualquer coisa do gnero. Teremos pessoas disponveis durante o evento inteiro para te ajudar com o que voc precisar!

Para que o ambiente seja bom para todas as pessoas muito importante que todo mundo esteja de acordo com o cdigo de conduta que estamos utilizando.
