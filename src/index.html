O que é Hacktoberfest?

O Hacktoberfest - trazido a você pela DigitalOcean em parceria com o GitHub e o Twilio - é uma comemoração de um mês de duração do software de código aberto. Os mantenedores são convidados a orientar possíveis colaboradores para questões que ajudarão a levar o projeto adiante, e os colaboradores terão a oportunidade de dar retorno aos projetos de que gostam e aos que acabaram de descobrir. Nenhuma contribuição é muito pequena - correções de bugs e atualizações de documentação são formas válidas de participação. Acesse o site oficial para se inscrever e saber mais.

Sobre o evento

Nesse encontro iremos explicar rapidamente sobre Git, Github e contribuições Open Source. Após as explicações cada um irá colocar a mão na massa e começar a contribuir com os projetinhos legais xD.

Para que o ambiente seja bom para todos é muito importante que todos estejam de acordo com o código de conduta.
